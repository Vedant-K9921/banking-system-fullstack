# Full Stack Banking System

A full stack banking application built using:

- Java
- Spring Boot
- MySQL
- HTML/CSS/JavaScript
- REST APIs

## Features

- User Registration
- Login Authentication
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- Responsive Dashboard UI

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- Bootstrap

Backend:
- Spring Boot
- Spring Data JPA
- REST API

Database:
- MySQL

## Run Project

Backend:
mvn spring-boot:run

Frontend:
Open login.html using Live Server
